    ToBoolEx(Object)

    The value passed as the first parameter is converted to a boolean value. If value is 0, null, false, NaN, DBNull, char '\0', char '0', the empty string (""), the string "false"(ignore case) or the string "0" will return false. All other values, including any object, will return true.
